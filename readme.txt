AJS(AngularJS) Instagram Feed displays impressive & responsive photos from instagram using AngularJS.
== Description ==This plugin will display photos from any non private instagram accounts to your website using AngularJS.	**Plugin Demo:** [AJS Instagram Feed Demo](http://www.appswifters.com/angularjs-instagram-feed/angularjs-instagram-feed-demo)> **Plugin Features**>> * Responsive Instagram Gallery> * Use Own Client ID> * Hide and show User Details> * Browser compatible> * Cutomize Button Text & Colors> * Hide and show Follow Button> * Display instagram on widget area> * Created using angularJS

== Installation ==
= Installation =
1. You can use the built-in installer.
     OR         
     Download the zip file and extract the contents.
     Upload the 'ajs-instagram-feed' folder to your plugins directory (wp-content/plugins/).
1. Activate the plugin through the 'Plugins' menu in WordPress.
Now go to **Settings** and then **AJS Instagram Feed** to configure any options as desired.
= How to use =
In order to display the instagram feed, you have three options
1. Simply place `<?php if (function_exists('ajs_instagram_feed_show')) echo ajs_instagram_feed_show(); ?>` in your theme
1. Add the shortcode as '[angularjs-instagram-feed]'
1. Using text widget & insert shortcode there.
